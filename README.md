### Description
- example dotnet 7 app proxying to [luceed](http://apidemo.luceed.hr)
- Learning C# 11 using dotnet 7.0.404 on debian 12

### Requirements
- dotnet, make

### Quick Start
```console
$ dotnet run --project=lucxy
```
or
```console
$ make run
```
